My Semantic HTML and CSS Cheat Page

HTML

Tag Name What It For
<header> Header Element represents introductory content, typically a group of introductory or navigational aids.
<article> Article Element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.
<footer> Footer Element represents a footer for its nearest sectioning content or sectioning root element.

CSS

Selector Example What It For
.class .name Selects all elements with class="name"
#id #name Selects the element with id="name"
element p Selects all <p> elements
